CSS 字体属性

字体系列

CSS 使用 font-famliy 属性定义 字体

字体大小

CSS使用 font-size 属性定义字体 大小

  • px（像素）大小是页面最常用的单位
  • 由于不同浏览器默认显示的字号可能不一致，所以尽量给一个明确值大小，不要默认大小

字体粗细

用 font-weight 定义字体 粗细

属性值 描述
normal 默认值（不加粗）
bold 定义粗体
100-900 400=normal,700=bold （后面不跟单位）

文字样式

font-style 属性定义文字风格

属性值 作用
normal 默认值
italic 斜体

总结

属性 字号
font-size 字号
font-family 字体
font-weight 字体粗细
font-style 字体样式
font 字体连写

CSS 文本属性

文本颜色

color属性定义文本的颜色

表示 属性值
预定义的颜色值 red,green,blue,pink
十六进制 #ff000,#ff6600···
RGB代码 rgb(255,0,0)或rgb(100%,0%,0%)

文本对齐方式

text-aligh 属性用于设置元素内文本内容的水平对齐方式

属性值 解释
left 左对齐（默认）
right 右对齐
center 居中对齐

文本装饰

text-decoration 属性规定添加到文本的修饰，可以给文本添加下划线、删除线、上划线等。

属性值 描述
none 默认，可以用来取消链接的下划线
underline 下划线
overline 上划线
line-through 删除线

文本缩进

text-indent 属性用来指定文本的第一行的缩进，通常是将段落 首行缩进


em 单位表示相对单位，就是当前1个文字的大小

行间距

line-height 属性用于设置行间的距离（行高），可以控制文字行与行之间的距离

上方距离

文字大小

下方距离

line-height 是文字上下方距离和文字大小之和

总结

属性 表示
color 文本颜色
text-align 文本对齐
text-indent 文本缩进
text-decoration 文本修饰
line-height 行高
