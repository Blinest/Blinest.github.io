对于表单的操作，vue 提供了 v-model 这个语法糖。

7.1. 基础用法

v-model 指令可以在表单 <input>、<textarea> 及 <select> 元素上创建双向数据绑定。它会根据控件类型自动选取正确的方法来更新元素。

它其实是一个语法糖，本质上包含两个操作：

  • v-bind 绑定一个 value 属性
  • v-on 给当前元素绑定 input 事件

注意：

v-model 会忽略所有表单元素的 value、checked、selected attribute 的初始值，而总是将 Vue 实例数据作为数据的来源。所以说我们需要在 data 选项中声明初始值。

v-model 在内部为不同的输入元素使用不同的 property 并抛出不同的事件：

  • text（文本） 和 textarea（多行文本） 元素使用 value property 和 input 事件；
  • checkbox（复选框） 和 radio（单选按钮） 使用 checked property 和 change 事件；
  • select（选择框） 字段将 value 作为 prop 并将 change 作为事件。
