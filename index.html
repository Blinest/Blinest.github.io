从数学角度看自由度

发表于 2022-03-11 更新于 2022-03-17 分类于 数学
本文字数： 358 阅读时长 ≈ 1 分钟

这段时间机械原理学了自由度，为探究其原理，就有了这一篇文章

前置知识

自由度计算公式：$F=3n-2P_L-P_H$

低副($P_L$)：移动副，转动副

高副($P_H$)：平面高副

自由度的判断的特殊情况：

复合铰链：两个以上的构件同在一处以转动副相连

局部自由度：不影响机构输出运动的自由度

虚约束：存在重复约束的机构

I级机构：机架和主动件构成

II级构件：2杆3低副

III级构件：4杆6低副

高副低代

数学解释

低副

约束掉2个自由度，也就是说只能做简单运动，平移或旋转

高副

约束掉1个自由度，也就是说在旋转的同时还可以进行平移，

自由度

对于空间构件：

自由度就是构件可以沿x,y,z轴移动和旋转，最多6个自由度

对于平面构件：

自由度就是构件可以沿x,y轴移动和平面旋转，最多有3个自由度

高副低代

一个高副其实可以用两个低副+一个活动构件来代替。

位运算

>> ：右移运算符，num>>1 相当于 num 除以 2

3.11

今天碰到了一个链表的小坑，如果链表的 node.next == null ，这里指向了就等于封闭了，没有下一个元素的地址了，如果这个时候 node = node.next 的话，那么就不能将其他链表挂到 node 上了，因为node.pre无法 找到它的地址，毕竟node.pre.next已经封闭了。

正确的做法应该是在 node.next ==null 时，如果想挂其他链表就马上把它的 next 挂到 node.next 上，这样就不会出现找不到地址的情况。
